## Rick and Morty Search

A single-page application for exploring characters from the Rick and Morty universe. Users can search for characters, browse result pages, and view detailed character profiles, providing an engaging and interactive data-driven experience.

[Check the Live Demo](https://ricky-and-morty-tau.vercel.app)

## Features
**Search and Filtering:** Implemented character search by name with dynamic query handling and real-time rendering of results.

**Pagination and Navigation:** Added pagination to efficiently browse large datasets and ensure smooth navigation across character lists.

**Character Detail Page:** Displayed extended information about each character, including status, species, origin, and episodes.

**State and Data Management:** Worked with a public REST API, implemented loading and error states to ensure stable user experience.

**Routing:** Used client-side routing to manage navigation between pages without reloading the application.
